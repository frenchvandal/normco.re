## Emojis en contexte

Les emojis s'integrent naturellement dans le texte :

> :bulb: **Astuce** : Utilisez les emojis avec parcimonie dans la documentation
> technique pour maintenir le professionnalisme tout en ajoutant de l'interet
> visuel.

### Indicateurs de statut

Les emojis sont parfaits pour les indicateurs de statut :

- :white_check_mark: Fonctionnalite implementee
- :construction: En cours de developpement
- :x: Non supporte
- :warning: Obsolete

### Style des messages de commit

De nombreux projets utilisent des emojis dans les messages de commit suivant des
conventions comme [Gitmoji](https://gitmoji.dev/) :

| Emoji                  | Type                    | Exemple                                                   |
| ---------------------- | ----------------------- | --------------------------------------------------------- |
| :sparkles:             | Nouvelle fonctionnalite | :sparkles: Ajouter le mode sombre                         |
| :bug:                  | Correction de bug       | :bug: Corriger le debordement de navigation               |
| :memo:                 | Documentation           | :memo: Mettre a jour le README                            |
| :art:                  | Style/format            | :art: Formater le code avec Prettier                      |
| :rocket:               | Deploiement             | :rocket: Deployer la version 2.0.0                        |
| :recycle:              | Refactorisation         | :recycle: Simplifier la logique d'auth                    |
| :lock:                 | Securite                | :lock: Corriger la vulnerabilite XSS                      |
| :globe_with_meridians: | Internationalisation    | :globe_with_meridians: Ajouter les traductions francaises |

## Accessibilite des emojis

Lors de l'utilisation des emojis, considerez l'accessibilite :

1. **Ne vous fiez pas uniquement aux emojis** pour transmettre un sens
2. **Utilisez du texte descriptif** avec les emojis
3. **Evitez la surcharge d'emojis** — trop peuvent distraire
4. **Testez avec les lecteurs d'ecran** — certains lisent les noms des emojis

## Resume

Les emojis sont un moyen amusant et expressif d'enrichir votre contenu.
Utilisez-les judicieusement pour ajouter de la personnalite sans compromettre la
clarte ou l'accessibilite.
