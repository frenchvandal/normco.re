# Source

This directory contains the Lume site source files used to build the static
site.

## Structure
- `_components`: Reusable HTML/TS components.
- `_includes`: Layouts, partials, and CSS bundles.
- `_data`: Shared data loaders.
- `_utilities`: Helper utilities used by templates.
- `pages`: Top-level content pages.
- `posts`: Blog post content.
- `js`: Client-side scripts.
