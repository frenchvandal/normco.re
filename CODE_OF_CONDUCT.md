# No Code of Conduct

This project has **no Code of Conduct**.

We believe that all contributors are capable, responsible adults who can
interact with one another in a civil and professional manner without the need
for a formal set of rules or enforcement mechanisms.

### What this means in practice

- Treat others with respect and courtesy.
- Focus on the quality of contributions rather than personal attributes.
- Resolve disagreements through direct, private communication whenever possible.
- If a participant's behaviour violates GitHub's Terms of Service or Community
  Guidelines, report it to GitHub directly.

We will not:

- Investigate or mediate interpersonal conflicts.
- Apply any form of social or ideological enforcement.
- Maintain a list of prohibited behaviours beyond what GitHub already prohibits.

Participation in this project implies acceptance of these expectations.

Thank you for contributing.
